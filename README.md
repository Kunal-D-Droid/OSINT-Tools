# OSINT Tools & Resources

A curated collection of **Open Source Intelligence (OSINT)** links for research, verification, and information gathering. OSINT is the practice of collecting and analyzing publicly available data—websites, archives, public records, and metadata—to build an accurate picture without privileged access.

> **Use responsibly.** These tools are intended for lawful research, security awareness, journalism, and personal due diligence. Respect terms of service, privacy laws, and local regulations. Do not use this list to harass, stalk, or harm others.

**Source of truth:** this README is the maintained list. `OSINT.txt` is a legacy text export from the original upload.
